<h1># 🎓 Rede Escolar - Sistema de Gestão</h1>


<h3>## 📌 Sobre o Projeto</h3>
O **Rede Escolar** é um sistema web desenvolvido em PHP para gerir a infraestrutura tecnológica de uma escola.  
Permite controlar salas, equipamentos, técnicos e intervenções de forma centralizada.

---

## 🚀 Funcionalidades

- 📊 Dashboard com estatísticas em tempo real  
- 🏫 Gestão de salas e técnicos  
- 💻 Inventário de equipamentos  
- 🛠 Registo e acompanhamento de intervenções  
- 🔎 Interface simples e intuitiva  

---

## 🛠 Tecnologias Utilizadas

- PHP  
- MySQL  
- Tailwind CSS  
- HTML5 + CSS3
